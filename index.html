Starting Git

Comandos essenciais do Git.

1. Inicializar um repositório Git
  1. git init

Este comando cria um novo repositório Git local.

2. Clonar um repositório Git
  1. git clone <url>

Este comando copia um repositório Git existente.

3. Adicionar arquivos ao índice
  1. git add <arquivo>

Este comando adiciona um arquivo ao índice para o próximo commit.

4. Commitar suas mudanças
  1. git commit -m "<mensagem>"

Este comando grava as alterações no repositório.

5. Push (enviar) suas mudanças
  1. git push <remoto> <ramo>

Este comando envia as alterações para o repositório remoto.

6. Pull (buscar) atualizações
  1. git pull <remoto>

Este comando busca e integra as alterações do repositório remoto.

7. Criar uma nova branch (ramificação)
  1. git branch <nome_da_branch>

Este comando cria uma nova branch.

8. Mudar para uma branch existente
  1. git checkout <nome_da_branch>

Este comando muda para uma branch específica.

9. Mesclar uma branch na branch atual
  1. git merge <nome_da_branch>

Este comando mescla a branch especificada na branch atual.

10. Ver o status do repositório
  1. git status

Este comando lista todos os arquivos modificados ou novos que ainda não foram commitados.
